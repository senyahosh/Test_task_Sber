# Тестовое задание Сбер
Выполненные тестовые задания (ML, SQL) для отбора на вакансию в Сбер

В первом задании на SQL был момент, из-за которого оно представлено в двух вариантах. В таблице user_roles имеется поле, которое я трактовал как дата, когда пользователю была присвоена его роль. Исходя из логики, до этой даты он не может совершать активности под этой ролью. У пользователя user6 в указанный период последнего месяца есть активности, но они, получается, выполнены без роли, так как дата этих активностей до дня назначения роли. В зависимости от того, было ли так задумано или же это просто несостыковка из-за того, что данные в таблице искусственные, возникает два варианта действий - учитывать дату назначения роли и не учитывать, отсюда и два варианта SQL запроса, которые дают чуть различающиеся ответы.
